# Fitness Coach API

## 📌 Description
A Flask-based web application that provides diet recommendations based on user input.

## 🚀 Features
- Diet plan suggestions
- JSON-based data handling
- Simple UI using HTML & CSS

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS
- JSON

## ▶️ How to Run
```bash
pip install -r requirements.txt
python app.py
